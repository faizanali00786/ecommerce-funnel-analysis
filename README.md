# E-commerce Funnel & Checkout Drop-off Analysis

## Business Objective
Analyze customer behavior across the e-commerce funnel to identify:
- conversion bottlenecks
- checkout drop-offs
- high-traffic low-conversion products
- session-level purchase behavior

---

## Tools Used
- SQL
- Power BI
- Excel

---

## Dataset Overview
- E-commerce behavioral event dataset
- Includes user interactions such as product views, cart additions, and purchases
- Analyzed customer sessions, product performance, and conversion behavior

---

## Key Analyses
- Funnel Conversion Analysis
- Strict Funnel Sequencing
- Product-Level Conversion Analysis
- Category-Level Conversion Benchmarking
- Session-Level Behavior Analysis

---

## Key Insights
- Significant drop-off observed between cart and purchase stages
- Certain high-traffic products showed disproportionately low conversion
- Funnel sequencing analysis identified friction during checkout progression
- Product-level analysis highlighted optimization opportunities for conversion improvement
